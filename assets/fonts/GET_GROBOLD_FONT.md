# Grobold Font — Setup

The game uses **Grobold** (by Guy Buhry) for all in-game text.

## Download
1. Go to https://www.dafont.com/grobold.font
2. Click **Download**
3. Unzip the archive
4. Copy `GROBOLD.ttf` into this folder (`assets/fonts/`)
5. **Rename it to `Grobold.ttf`** (capital G, matching what the code expects)

The game will fall back to Godot's default font if the file is missing,
so everything still runs without it — it just won't look right.
